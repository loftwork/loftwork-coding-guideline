# 5-3. 記述ルール <Badge type="danger" text="必須" />

## 5-3-1. 文字コード

CSSの文字コードはHTMLの文字コードに合わせ、1行目に記述します。

```css
@charset "UTF-8";
```

## 5-3-2. 単位・値の省略

- 値が「0」の場合、単位は省略する（`margin: 0;`）
- 小数点の頭の「0」は省略してよい（`.8em`）
- HEX形式のカラーコードで3文字で表記できるものは省略してよい（`#ebc`）

## 5-3-3. プロパティの終端

すべてのプロパティ値の終端にはセミコロンを記述します。プロパティ名の後にはコロン＋スペースを入れます。

::: code-group

```css [NG]
.test {
  display: block;
  height: 100px
}
```

```css [OK]
.test {
  display: block;
  height: 100px;
}
```

:::

## 5-3-4. 要素の可視性

仕様上避けられない場合を除き、すべてのテキストはユーザーに可視化させます。隠しテキストとなるような表現はしてはなりません。

> **NG例：** `text-indent: -9999px`、`font-size: 0`、背景色と `font-color` が同色または類似色

## 5-3-5. CSSハック

CSSハックは極力使用しません。
