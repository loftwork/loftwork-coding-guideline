# 7-5. ページタイトルと言語 <Badge type="danger" text="必須" />

## 7-5-1. ページタイトル（達成基準 2.4.2）

各ページの `title` 要素には、ページの内容を適切に表すタイトルを記述します。「ページ名 | サイト名」の形式を推奨します。

::: code-group

```html [NG]
<title>トップページ</title>
```

```html [OK]
<title>採用情報 | ロフトワーク株式会社</title>
```

:::

## 7-5-2. 言語コードの指定（達成基準 3.1.1）

`html` 要素には `lang` 属性で主要な使用言語を指定します。

```html
<html lang="ja">
```
