# 7-6. フォームのアクセシビリティ <Badge type="danger" text="必須" />

## 7-6-1. ラベルと入力欄の関連付け（達成基準 1.3.1 / 3.3.2）

フォームのラベルとコントロールは `for` / `id` 属性で関連付けます（[4-4. フォーム](/04-html/4-4-forms) と連動）。エラーメッセージは対象の入力欄と関連付けてプログラム的に表現します。

## 7-6-2. エラーの特定と提案（達成基準 3.3.1 / 3.3.3）

入力エラーが検出された場合は、どの項目にエラーがあるかをテキストで提示します。また、修正方法の提案も可能な範囲で提供します。
