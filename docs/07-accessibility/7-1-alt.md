# 7-1. テキストの代替

本ガイドラインでは WCAG 2.2 レベルA の達成を最低基準とします。

本章で扱う要件は、コーディングおよびマークアップで担保可能な範囲を対象とします。情報設計・画面設計など実装以外で主に担保される要件の扱いは、[はじめに](/intro#アクセシビリティ項目の扱い) を参照してください。

## 7-1-1. alt属性の付与（達成基準 1.1.1） <Badge type="danger" text="必須" />

`img` 要素・`area` 要素・`input[type=image]` などの非テキストコンテンツには必ず代替テキストを付与します。

- **意味を持つ画像：** 内容を表すテキストをalt属性に記述する
- **装飾目的の画像：** alt属性を空（`alt=""`）にする

::: code-group

```html [NG]
<img src="logo.png">
<img src="banner.png" alt="画像">
```

```html [OK]
<img src="logo.png" alt="ロフトワーク">
<img src="deco.png" alt="">
```

:::

## 7-1-2. 動画・音声コンテンツの代替（達成基準 1.2.1〜1.2.3） <Badge type="warning" text="推奨" />

収録済みの動画・音声コンテンツには、キャプションまたはテキストによる代替を提供します。
