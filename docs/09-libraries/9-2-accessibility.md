# 9-2. WCAG/アクセシビリティ対応 <Badge type="warning" text="推奨" />

モーダル・カルーセル・ドロップダウンなどのUIコンポーネントライブラリは、アクセシビリティ対応の有無が実装品質に直結します。以下の点を確認してください。

- 公式ドキュメントにアクセシビリティ（WAI-ARIA対応・キーボード操作・スクリーンリーダー対応）に関する記述がある
- WCAG 2.2 レベルAまたはAAへの準拠が明示されている
- `aria-*` 属性・`role` 属性・`tabindex` が適切に実装されている

> ライブラリ側がアクセシビリティを謳っていても、利用側の実装（色・コントラスト・テキスト代替など）で担保すべき項目は別途対応が必要です。
