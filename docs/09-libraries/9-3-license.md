# 9-3. ライセンス <Badge type="danger" text="必須" />

クライアント案件への適用を前提とするため、商用利用が許可されているライセンスであることを必ず確認します。

**利用可能なライセンスの例：**

- MIT License
- Apache License 2.0
- BSD 2-Clause / 3-Clause License
- ISC License

**確認が必要なライセンス：**

以下は商用利用に制限がある場合があるため、導入前に必ずLW担当者に確認します。

- **GPL（GNU General Public License）系：** 派生物のソースコード公開義務が生じる場合がある
- **CC（Creative Commons）系：** ライセンスの種別により商用利用禁止や改変禁止のものがある
- **独自ライセンス：** 内容を精査し、必要に応じて法務確認をおこなう

> ライセンスはパッケージの `LICENSE` ファイルまたは `package.json` の `"license"` フィールドで確認できます。
